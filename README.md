# cyclistic-bike-share-analysis

This project analyzes usage patterns of a fictional bike-share company, Cyclistic, to understand the differences between casual riders and annual members.

The goal of the analysis is to generate insights that can help the company design marketing strategies to convert casual riders into annual members.

This project is part of the Google Data Analytics Professional Certificate capstone case study.

Business Task - How do annual members and casual riders use Cyclistic bikes differently?

The insights from this analysis will help the marketing team create targeted campaigns to encourage casual riders to purchase annual memberships.
